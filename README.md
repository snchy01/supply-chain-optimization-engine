# Supply Chain Optimization Engine
### End-to-End Data Pipeline & Prescriptive Analytics on Databricks

**Project Overview:** This project demonstrates a production-grade data pipeline designed to manage and optimize inventory for **100,000+ SKUs**. By integrating **Big Data processing** (PySpark) with **Operations Research** (Linear Programming), the engine identifies stock-out risks and mathematically allocates a limited budget to minimize operational impact.

## ðŸ›  Technical Architecture
The project follows the **Medallion Architecture** to ensure data quality and reliability:

* **Bronze Layer (Raw):** Synthetic generation of 100,000+ supply chain transaction logs and inventory records.
* **Silver Layer (Cleansed):** Data transformation using **PySpark Window Functions** to calculate **ABC Categories**, **Stock-out Risk Scores**, and **Lead Time Variability**.
* **Gold Layer (Optimized):** A final "Procurement Plan" table, generated by a mathematical solver, providing prescriptive order quantities.
